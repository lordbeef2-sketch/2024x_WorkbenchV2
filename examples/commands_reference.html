Quick Start: Teamwork Cloud REST

Use the shared auth helper once, then pass the client into the REST commands. If a command needs IDs like resource_id or branch_id, it can often pull them from examples/config.json when you omit them.

Usage
from Modules import build_authenticated_client
from Modules.commands import list_workspaces, list_models

client = build_authenticated_client()
workspaces = list_workspaces(client)
models = list_models(client, resource_id="RESOURCE_ID", branch_id="BRANCH_ID")

Quick Start: Local No Magic 2022x

The No Magic commands work against a local MagicDraw or Cameo installation through JPype. Reuse the same config or API object across calls when you are making several requests in one script.

Usage
from Modules import MagicDrawJVMConfig
from Modules.commands import get_active_project, get_project_summary, list_project_models

config = MagicDrawJVMConfig.from_environment()
project = get_active_project(config=config)
summary = get_project_summary(project, config=config)
models = list_project_models(project, config=config)
