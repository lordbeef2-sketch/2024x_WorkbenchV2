2024x Refresh3 Version News
  • Created by Kristina B., last updated on Jul 04, 2025 3 minute read

Released on: July 11, 2025


The 2024x Refresh3 release introduces key enhancements to collaborative modeling on both Teamwork Cloud and the 3DEXPERIENCE platform. Teamwork Cloud now supports partial project usages as a technology preview, enabling more flexible reuse of specific project parts, along with improved SSO support across command-line tools. On the 3DEXPERIENCE platform, standard profile migration has been simplified, and project opening performance with local cache has been significantly improved. Both Teamwork Cloud and the 3DEXPERIENCE platform will benefit from a new option to detect changes in derived properties during project merging or comparison.

To download the latest version of the modeling tool, see Downloading installation files. For further information, check the product documentation.

Collaboration

Teamwork Cloud

Partial Project Usages (Technology Preview)

We’re excited to introduce a significant enhancement to project usages, now available as a technology preview. Previously, you could only use an entire project (full usage), but with this release, you can define a usage scope and use only a specific part of a project. Additionally, you can seamlessly switch between full and partial usage at any time and adjust the usage scope as needed.

Selecting the scope of a partial project usage.

Improvements to Single Sign-On Authentication

For improved security, the Used Projects Auto Update Wizard now only accepts access tokens generated by your user when using single sign-on (SSO) authentication. Additionally, SSO is now supported in other command-line tools, like Report Wizard and Cameo Collaborator.

3DEXPERIENCE Platform

Easier Standard Profile Migration

The standard profile migration has been simplified allowing you to migrate all standard profiles in one go. Now, you can use the project migration dialog to select the profiles from both the server and your modeling tool installation. During migration:

  • Standard profiles stored in the 3DEXPERIENCE platform are migrated.
  • Standard profiles existing in the modeling tool installation but not yet added to the the server can be selected and added to the 3DEXPERIENCE platform.
  • Any dependent profiles are also migrated or added.

Performance Improvements

In version 2024x Refresh3, project opening performance has been significantly improved when a local cache is already present on your machine, as demonstrated in the chart below.

Comparison of project load times with local cache.

Other Improvements

  • The following predefined templates have been introduced for publishing Cameo Collaborator documents:
    • UAF Template for Cameo Collaborator
    • UPDM Template for Cameo Collaborator
    • BPMN Template for Cameo Collaborator
  • You can now specify the project branch, revision, and iteration when logged in via the command line interface.

General Collaboration Improvements

  • The new Detect Changes in Derived Properties environment option allows you to specify which derived properties to compare when detecting changes during project merge or compare.
